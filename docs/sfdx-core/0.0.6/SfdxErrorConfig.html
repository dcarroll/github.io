SfdxErrorConfig

A class to manage all the keys and tokens for a message bundle to use with SfdxError.

Constructor

new SfdxErrorConfig(bundle, errorKey, errorTokens, actionKey, actionTokens)

Source:
  • sfdxError.js, line 18
Create a new SfdxErrorConfig.
Example
SfdxError.create(new SfdxErrorConfig('apex', 'runTest').addAction('apexErrorAction1', [className]));
Parameters:
Name Type Description
bundle The message bundle.
errorKey The error message key.
errorTokens The tokens to use when getting the error message
actionKey The action message keys.
actionTokens The tokens to use when getting the action message(s)
